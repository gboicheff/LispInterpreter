# Lisp Interpreter


## How to use it

### Run a code file

1. Put a lisp file in the codefiles folder

2. Run the following command:

```
python run_file.py filename
```

### Run a repl
Run the following command:

```
python run_repl.py
```

### Run tests
Run the following command:

```
python run_tests.py
```


## Functionality
* Loops (while, dotimes)
* Variables and Scoping (defvar, setq, let)
* Functions (defun)
* Conditional (if)
* Lists (list)
* Small standard library
